# n8n-nodes-dingbot

Dingbot nodes for [N8N](https://n8n.io/).

## About Dingbot

Dingbot is a [DingTalk](https://dingtalk.com/) bot that can be used to send notifications to your team.

## Installation

```sh
npm install n8n-nodes-dingbot
```
